By installing the "H3Plugin.Template.vsix" visual studio extension included in this folder,
you will have a quick way to create your own plugins within the H3Plugin solution with only a few button clicks.

Alternatively, place the "H3Plugin.Template.zip" archive in your Visual Studio templates folder.
